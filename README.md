# watsonx_code_assitant_for_i-Private_Preview-TestCase
Source codes for public preview of watsonx code assistant for i

![The test cases used the same source as the “Private Preview”.](TestCases.png)

## プロジェクトの概要

このプロジェクトは、watsonx code assistant for i の機能を検証するためのソースコードを提供します。IBM i環境でRPG言語を使用しており、様々なプログラムが含まれています。

## 使用技術

*   RPG (RPGI2A_FreeFormat.RPGLE)
*   ILE
*   SQLRPGLE
*   データベースファイル (.PF, .LF)

## セットアップと実行方法

このプロジェクトを実行するには、IBM i環境が必要です。具体的なセットアップ手順については、関連ドキュメントを参照してください。
