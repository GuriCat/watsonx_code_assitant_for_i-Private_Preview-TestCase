# AGENTS.md (Code Mode)

This file provides guidance to agents when working with code in this repository, specifically in Code mode.

## RPGLE Specifics

- Use free-format RPGLE as defined in `RPGI2A_FreeFormat.RPGLE`.
- SQLRPGLE files (`DSPSVRSTSR.SQLRPGLE`) embed SQL within RPGLE.
- Be mindful of the mix of traditional and SQLRPGLE code.
